内容电商相关

发表于 2018-12-17

一.目前的内容电商

说起内容电商，目前主流的内容电商主要分为两类：

  • 以精选内容为主，通过社区的形式引导一部分用户产生内容，普通用户对于一部分商品知识有获取的需要，进一步引导转化。比如小红书。
  • 以IP为主，通过内容让用户对IP产生信任，满足用户获取知识的需要，基于对IP的信任完成转。比如年糕妈妈。

淘宝从很早之前就开始做内容电商，最早的淘女郎，现在的微淘，都是为了满足用户逛的需求。其理念是，用户把时间花在这里了，总会买东西的。如果非要归类，我会把他归为第二类，用户基于对商家或品牌的信任，获取内容和购买商品。

二.字节系的内容

目前字节系的产品，是很纯粹的内容产品基因，无论是头条还是抖音，其特点都是依靠内容本身吸引流量。
所以结果上来看（粉丝触达率10%），IP概念比较弱，第二条路子天然不好走。
而第一条路子，抖音是一个泛娱乐场景，头条是一个资讯场景，交易的属性天然就很弱。结果上来看的问题便是转化率低，即使流量很大最后成单也有限。进一步地，作为达人商品相关的内容创作意愿度不高，作为商家创作成本高ROI低，不能持续产出优质的商品内容。

三.用户对于消费商品内容的真实需求

我们深陷局中，总以为我们有流量，总以为我们有内容，其实我们最缺精准流量，最缺电商优质内容。而其实我们应该跳出来，对于商品内容，用户真正需要的是什么，内容对于用户购物解决什么问题。对于商品相关内容，我认为主要有以下几类信息是用户需要的。

  • 好看–对于美丽的需求–服饰、美妆
  • 好吃–对于好吃的需求（味觉&健康）–生鲜、零食、养生食品
  • 好用–对于便捷的需求–百货
  • 好玩–对于新鲜的需求–新奇特
  • 知识–对于自我提升的需求–虚拟产品
  • 便宜–对于金钱的需求–标准品类

根据马斯洛的需求理论，越基础的需求共性越大，比如便宜，本质上是用户对财产安全的需求，是除了基础的生理需求外最大的需求。
所以微信中的几个品类，服饰、美妆、家居百货、食品，本质上切中的是用户的这几部分需求，只是传播的方式更多是通过社群，传播的内容会随意化一些。其本质都是利用了用户以上的某种需求，内容给予了用户信息增益。

四.用户对于购买商品的真实需求
我们天然的以为，用户看了文章，会自然地去购买文章中的商品。但实际上，消费内容决策和购买决策是两个不同的环节。比如，关于iphone如何好用的内容，我有兴趣阅读，但是买不买，还取决于你的价格，你的服务，我的购买力，我对你的信任。所以用户对于购买商品的真实需求，并没有因为内容而改变。
形成一次购买的核心要素还是

  • 需求（用户真实的需求）
  • 信任（持续的服务能力）
  • 价格（价格在市场上的竞争力）
  • 购买力（用户的真实购买力）

所以从结果上来看，广义的内容电商（包含微商），卖的好的以非标品和相对价低的标准品为主。因为从内容到购买的衔接，衔接好的无外乎以下两类。

内容有信息增益（变得好看／变得聪明，产生需求），商品非标（信息不对称）
内容传递的信息就是便宜，商品标准（信息对称）

解决的都是需求和价格的问题，信任需要持续稳定的服务去积累（心智）

五.大值点的内容电商

说了那么多，其实内容电商能做什么已经很清晰了。大值点的内容电商应该是基于整个字节跳动的内容生态（不再质疑），对电商内容的聚合和分发，并通过内容解决用户的某个共性需求，通过商品完成转化。分发态下的去中心化电商触发用户的某个需求，聚合态下的中心化场景集中解决用户某个需求。

  • 聚合共性需求的优质内容，并有机制可以持续产生（什么值得买／小红书／下厨房）
  • 能够产生优质内容并解决共性需求的达人，打造ip（蘑菇街）
  • 高性价比的标准品（天猫超市）
    去中心化消费场景：非紧急购物需求，内容分发后的触发式购物
    中心化消费场景：这里不只是购物，更是解决需求的地方
    核心优势：用户的时间在这里，我有机会通过内容最先触达你
    愿景：消除信息不对称，让原始需求和真实购买力碰撞

六.case

  • case1：抽纸作为家庭消费品，是家里日常需要的商品。我不会主动想起要去天猫买个纸（非缺失情况），但是头条抽纸做活动的信息触达我，我会选择点进去购买。
  • case2：我想变得好看，我不会主动打开淘宝去看大量的商品然后购买，但是我看到抖音某个人的穿搭好看，我会选择点进去购买。

对拼多多的十次思考

发表于 2018-10-29

记得第一次了解拼多多是在16年的7月，而如今其在纳斯达克上市也已经过去两个月了，这中间，不过两年而已。两年来，拼多多一直饱受争议，有批判其卖假货的，产品没有下限的，也有抄袭他流量获取方法的，有褒扬他供应链效率的。而我恰恰对其保持了持续的关注，期间产生了一些思考，分享给大家。

一.模式新颖

第一次认识拼多多的时候，那会刚毕业不久，对电商的了解也实在有限。仅仅知道拼多多在微信里面利用拼团的方式，将拉新的成本让利给了用户，最终实现了以较低的成本实现了获取新客。对拼多多的理解停留在创意新颖。

二.货品瓶颈

第二次思考拼多多的时候，已经对电商有了一定理解，认为货品是一个电商的核心。而拼多多留给我的标签是“价格便宜”、“质量差”，靠罚商家的钱盈利。我认为货品质量差的话，早晚会制约其发展，货品会成为增长的瓶颈，拼多多会陷入到便宜的东西质量不好，质量好的东西不便宜，价格贵的东西对用户没有吸引力，质量差的东西会伤害用户的怪圈中去。靠罚商家的钱盈利的商业模式不可持续，当时是不看好拼多多的。

广告思维与零售思维

发表于 2018-09-30

思考

  • 广告思维是在一次pv下，通过各环节的优化提升转化率，获得产出。所以广告商品更关注卖点（适合营销）和利润（获取pv）
  • 零售思维是在一次购物中给用户留下好的印象（确定性）。所以电商商品更关注品类（复购场景）和体验（复购可能性，包括价格）
  • 电商产品要在广告思维和零售思维下取得平衡点

case

  • 广告爆品在取得较好的转化后，没有产生较好的复购，流量依赖性大。用户看到商品了觉得不错会购买，但是不会主动想起来要来买东西

特征值检验

发表于 2017-06-16

特征检验

卡方检验

特征/结果 R 非R 总计
A a b a+b
非A c d c+d
总计 a+c b+d a+b+c+d

令样本数N=a+b+c+d

假设A与R无关，即A＝非A

则R的概率应该是(a+c)/N

那么得到期望：

特征/结果 R 非R 总计
A (a+c)/N*(a+b) (b+d)/N*(a+b) a+b
非A (a+c)/N*(c+d) (b+d)/N*(c+d) c+d
总计 a+c b+d a+b+c+d

根据卡方检验的公式: 

D = ( ∑(E1-E0)² )/E0，E1是实际值，E0是理论值（期望值）

D越大（A,R）越相关，D越小(A,R)越不相关

KA商家运营

发表于 2017-04-06

B2C的平台(bussiness to customer)，顾名思义，核心业务是通过平台把商家的货(B端)直接卖给消费者(C端)。所以这一类的企业，不管是淘宝，还是其他类似的平台，商家运营的工作就显得十分重要，因为对于消费者来说，平台的核心价值就是货，而货都是商家提供的。此外，商家运营，因为在划分组别的时候大都按照品类规划，所以很多地方也会叫做类目运营，行业运营，品类运营，但是核心工作其实都是商家运营。

就像很多企业会圈定自己的核心用户一样，受限于人员精力以及运营成本，电商运营的过程中也对自己的平台的商家去圈定核心商家进行重点运营，主要工作有如下几点:

  • KA商家选择，建立KA商家池
  • KA商家运营（合作，推广，培养）
  • KA商家迭代

写在数据分析最前面

发表于 2017-04-05

往往我们谈到数据分析，或者说想要从事数据分析工作，自然而然地，我们首先会想到R,Python等编程语言，亦或是决策树，神经网络等分析算法。在大数据概念火热的年代，”啤酒与尿布”的神话给数据分析披上了一层高贵的面纱。然而在我看来，数据分析并非如此高贵冷艳，人人都可以成为数据分析师，因为我们追寻本质，数据分析无非就是解决问题的一个途径罢了。而最终影响你的高度的，可能是你对这个行业的理解深度，也就是你对这个行业的问题的理解深度。所以写在数据分析最前面，我认为我们始终要聚焦到以下几点：

  • 面向什么用户什么场景，遇到了什么问题
  • 需要用到什么样的数据
  • 对数据进行了怎样的处理和分析
  • 通过数据得到了什么结论
  • 得到的结论带来了什么价值
