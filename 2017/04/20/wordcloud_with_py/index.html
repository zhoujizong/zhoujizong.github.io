python实现词云绘制

发表于 2017-04-20

主要工具

  • python 2.7
  • jieba分词
  • wordcloud绘制词云

github 地址

  • https://github.com/fxsjy/jieba/
  • https://github.com/amueller/word_cloud/tree/master/wordcloud

工具安装

  • pip install jieba
  • pip install wordcloud
  • 也可以下载文件后运行setup.py进行安装

example–以淘宝的差评为例

背景

假如你是一个商家，想要看下自己店铺内的商品差评情况，来确定用户对本店商品不满意的点主要有哪些，针对性地去改进，那么我们可以通过对每一条差评进行分词提取关键词，然后再绘制一个词云，就可以很直观地看到用户不满意的key point了。

利用jieba分词对差评进行分词

# -*- coding: utf-8 -*-
import jieba.analyse

#利用tf-idf算法的分词函数，输入文本和需要的top重要的关键词个数
def cutting(content,topK):
    tags = jieba.analyse.extract_tags(content, topK=topK)
    return ','.join(tags)

file_name_r = open('./data/input_file','r')
file_name_w = open('./data/output_file.txt','w')

#print cutting('eric是个好孩子',2)

for line in file_name_r:
    line = line.decode('utf-8')
    cutting_out_u = cutting(line,2)
    cutting_out_s = cutting_out_u.encode('utf-8')
    file_name_w.write(cutting_out_s+'\n')

file_name_r.close()
file_name_w.close()

利用wordcloud对分词好的词绘制词云

import matplotlib.pyplot as plt
from wordcloud import WordCloud
import jieba

text_from_file_with_apath = open('./data/output_file.txt').read()

wordlist_after_jieba = jieba.cut(text_from_file_with_apath, cut_all = True)
wl_space_split = " ".join(wordlist_after_jieba)

my_wordcloud = WordCloud().generate(wl_space_split)

plt.imshow(my_wordcloud)
plt.axis("off")
plt.show()

结果输出

所以我们可以看到商品质量问题是首要问题，另外客服，物流次之。

注意点

  1. 注意数据存储路径，我是放在当前目录的./data/下面
  2. 注意中文编码问题，这个博主写的挺清楚的：http://www.cnblogs.com/baihuitestsoftware/articles/5230351.html
  3. wordcloud默认用的英文字体，所以我们在绘制中文词云的时候可能会出现一个个框，可以打开python包wordcloud的安装路径，我们直接看源码，wordcloud.py默认设置了:
    FONT_PATH = os.environ.get(“FONT_PATH”, os.path.join(os.path.dirname(file),”DroidSansMono.ttf”)
    把DroidSansMono.ttf更改为自己想要的字体即可。我用的是msyh.ttf，需要自己下载，放到你的脚本目录下。
  4. 分词的算法以及词云的形状颜色均可配置，自己根据需要选择哈，本文因为是以基本实现词云绘制为主，所以不再拓展。
