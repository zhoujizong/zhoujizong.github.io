特征值检验

发表于 2017-06-16

特征检验

卡方检验

特征/结果 R 非R 总计
A a b a+b
非A c d c+d
总计 a+c b+d a+b+c+d

令样本数N=a+b+c+d

假设A与R无关，即A＝非A

则R的概率应该是(a+c)/N

那么得到期望：

特征/结果 R 非R 总计
A (a+c)/N*(a+b) (b+d)/N*(a+b) a+b
非A (a+c)/N*(c+d) (b+d)/N*(c+d) c+d
总计 a+c b+d a+b+c+d

根据卡方检验的公式: 

D = ( ∑(E1-E0)² )/E0，E1是实际值，E0是理论值（期望值）

D越大（A,R）越相关，D越小(A,R)越不相关

信息熵

根据信息熵公式: 

H（A）= -∑ ( P(A)*log(P(A)) )

在A未知的情况下，总体样本的信息熵为:

H(1) = -(a+c)/N*log((a+c)/N)-(b+d)/N*log((b+d)/N)

而在A的情况下，样本的信息熵为：

H(2) = -a/(a+b)*log(a/(a+b))-b/(a+b)*log(b/(a+b))

信息增益为H(2)-H(1),增益越大则越相关。

互信息

根据互信息公式：

I(R;A) = ∑P(R)*log(P(A|R)/P(A))

所以可以得到A和R的互信息为

I(R;A) = (a+c)/N*log((a/a+c)/((a+b)/N))+(b+d)/N*(log(b/b+d)/((a+b)/N))
  • I >> 0,越正相关
  • I ≈ 0,越不相关
  • I << 0,越负相关

频率比较法

最后，其实还有一个比较简单的方法，可以直观上帮助我们判断

P(A|R) = a/(a+c)
P(A|非R) = b/(b+d)

如果P(A|R) >> P(A|非R)，那么可以认为A与R相关
